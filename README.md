# ollama-playground

A little script to demo ollama tooling.

## Running

You will need [uv](https://docs.astral.sh/uv/) installed before proceeding.

* Run `uv sync` to sync dependencies
* Run `uv run code_analysis_pro.py <directory to scan>` to run the script on the provided directory and get an answer to what the qwen model thinks the code repo is about.
